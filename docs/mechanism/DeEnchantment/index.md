# 负魔书

负魔书是一款趣味插件，可以让你在附魔时得到与原版效果的反向附魔，也就是负魔。

原版的附魔太无聊了？那么就来尝试一下打造真正的极品装备吧！

## 触发条件及使用方法
负魔将会在以下事件发生时尝试出现：

* 使用附魔台附魔时
* 开启未打开过的战利品箱子时
* 怪物自然生成时
* 村民出现交易选项时
* 钓鱼钓到附魔书时
* 猪灵交易、村庄英雄使村民丢附魔书时

以下工具可以对负魔进行操作：

* 使用铁砧将负魔应用于物品上
* 使用砂轮将负魔去除
